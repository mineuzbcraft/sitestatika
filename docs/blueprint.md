# **App Name**: FuqaroPortal

## Core Features:

- User Authentication & Registration: Provides secure login for both users and administrators. Allows new users to register with required fields such as full name, email, auto-masked phone number, validated password, and a selection of Tashkent districts, utilizing client-side storage mechanisms for MVP.
- Modular Interface Layout: A dynamic application layout featuring a persistent top navigation bar (including app logo, logged-in username, and a logout option) and a collapsible left sidebar presenting clickable icons for all seven distinct modules, designed for seamless navigation.
- DoriInfo (Medicine Search & Management): Enables users to search for medicines, apply category filters, and view detailed information including name, dose, and potential side effects. Includes dedicated administrative tools for adding, editing, and deleting medicine entries, managed through client-side data.
- Ariza (AI-Powered Letter Generator): An intelligent tool that guides users through generating formal Uzbek application letters. Users select from eight application types, input details via form fields, and an AI tool processes this input to compose a grammatically correct and formally structured letter, with options to copy or print. Administrators have access to review all generated letters.
- Interactive Modules (University & Rights): Integrates two interactive modules: 'Tanlayuni' allows users to find matching universities based on subject interests and DTM exam scores via a slider, with admin capabilities for data management. 'Huquqim' provides access to youth rights information across six categories with fifteen detailed cards based on Uzbek law references, a search bar, and essential emergency contacts, also with admin data management features.
- Mahalla Muammo (Problem Reporting & Geo-tracking): Allows users to submit local problem reports including district, type, and description. Displays reported problems with dynamic status badges. Features an interactive map (Leaflet.js) to visualize problem locations. Administrators can update problem statuses and access visual statistics (Chart.js) related to reported issues.
- Career Guidance & Admin Dashboard: Offers the 'Kasbim' module, a 15-question career test with a progress bar that provides personalized results, suggesting top three career paths, potential salaries, and relevant universities. Also includes a comprehensive Admin Dashboard featuring key statistical cards (users, problems, applications, tests), a Chart.js bar chart for module usage, and a list of recent activities.

## Style Guidelines:

- Primary color: A vibrant purple (#7B2FF7), used for primary interactive elements, branding, and emphasis, contributing to a modern and authoritative aesthetic within the dark theme.
- Page Background: A dynamic gradient transitioning from a deep indigo (#0F0C29) to a rich violet (#302B63), establishing a sophisticated and immersive dark foundational visual for the entire application.
- Element Background: A deep, desaturated charcoal-violet (#1D1A1E), harmonizing with the primary hue while offering subtle contrast for UI components like glassmorphic cards and section backgrounds, adhering to the dark theme.
- Accent Color: A vivid electric cyan (#00F5FF), strategically applied to highlight critical information, active states, and call-to-action elements, ensuring strong visual contrast and digital vibrancy.
- Body and Headline font: 'Inter' (sans-serif), chosen for its exceptional readability, modern aesthetic, and versatility across both prominent headlines and detailed body text within the portal. Note: currently only Google Fonts are supported.
- Minimalist, crisp line icons rendered primarily in the accent cyan for module navigation, notifications, and interactive elements, designed to enhance user recognition and maintain a clean, high-tech glassmorphism appearance.
- Responsive layout featuring a fixed top navigation bar and a collapsible left sidebar. Content areas within modules will adopt glassmorphic cards, utilizing a transparent white background (rgba(255,255,255,0.1)) and a strong backdrop blur (20px), ensuring mobile responsiveness.
- Subtle and fluid animations will be applied to UI elements upon interaction (e.g., hover, click), including smooth transitions for module changes, state updates (like problem statuses), and clear, yet non-intrusive toast notifications.
